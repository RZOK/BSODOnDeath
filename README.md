# BSODOnDeath
A tModLoader mod for Terraria 1.3.5 that causes your PC to bluescreen whenever you die.

This is not a joke.

It will ***actually*** bluescreen your computer.

"Why?" -90% of r/Terraria

Does not work on Mac or Linux.
